# Networth Tracker

A small personal net worth tracker built with React and Vite.

Live preview (after deploy) will be available at:

https://limqihui08.github.io/networth-tracker/

## Run locally

```bash
cd 'C:\Users\user\Downloads\networth'
npm install
npm run dev
# open http://localhost:5173/
```

## Build

```bash
npm run build
```

## Deploy (GitHub Pages)

This repository contains a GitHub Actions workflow that builds the site and deploys the `dist/` folder to the `gh-pages` branch on push to `main`.

If Pages isn't enabled automatically, go to the repository Settings → Pages and select the `gh-pages` branch as the source.
